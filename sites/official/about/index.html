What is NeoBrowser?

NeoBrowser is a custom lightweight local web browser built in Python using PyWebView and Tkinter, designed to explore .neo domains and fetch:// URLs stored in a local registry.

It combines a native GUI interface with an embedded browser engine, providing a fast, responsive, and modular browsing experience without relying on heavy browsers.

How to Use NeoBrowser

Launch Browser

Opens with a blank page ready to search.

Search

Type a keyword or .neo domain in the top search bar. Press Enter or click Go.

View Results

Double-click a search result to open the page in the embedded browser. Search bar stays visible for repeated queries.

Add New Sites

Update registry.json with new domains and local HTML paths.

// Example registry.json entry
{
  "example.neo": {
    "name": "Example Site",
    "path": "/path/to/local/file.html"
  },
  "fetch://example": {
    "name": "Example Fetch Site",
    "path": "/path/to/another/file.html"
  }
}
